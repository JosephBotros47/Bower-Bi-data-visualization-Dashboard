# Power BI Data Visualization Dashboard

## 📌 Project Overview
This project focuses on transforming raw data into an interactive and visually appealing dashboard using **Power BI**. The primary goal was to create a comprehensive analytical tool that allows users to gain insights and monitor key metrics effectively.

## 📊 Data Source
- **Format:** Microsoft Excel (.xlsx)
- **Status:** The project utilizes a pre-existing dataset. 
- **Note:** In this specific project, the focus was placed on **Data Modeling** and **Visualization Design** rather than Data Cleaning (ETL). The dashboard was built directly using the provided structured data to showcase design and reporting capabilities.

## 🚀 Key Features
- **Interactive Visuals:** Dynamic charts and graphs that respond to user selections.
- **KPI Tracking:** High-level summary of essential business or data metrics.
- **Slicers & Filters:** Ability to drill down into specific categories or timeframes for deeper analysis.
- **User-Centric Layout:** A clean, professional interface designed for easy navigation.

## 🛠️ Tools Used
- **Power BI Desktop:** For report building and dashboard creation.
- **Microsoft Excel:** As the primary data source.
